# others-ui

## How to run ？

```shell
pnmp i
pnpm dev
```



## 开发规范

1. main分支切出去
2. 本地dev根目录运行`pnpm i && pnpm dev` (注意node版本大于16)
3. lit组件开发目录packages/others
4. 开发完成后提交到远程向main发起pr
5. 后续等项目发布第一版再完善
